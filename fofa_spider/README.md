# Pool man's Fofa Spider  :zap:
## Selenium模拟操作Chrome浏览器爬取Fofa检索结果   
适用于和我一样买不起会员账户,用不上Vip Api,但是还想简单快捷通过Fofa采集信息的"穷逼"  
## 使用  
**需要自己辨认验证码并且点击登录！**  
修改账号密码   
修改检索用的Key  
修改chromedriver的路径  
## 需要安装的库  
selenium  
urllib
